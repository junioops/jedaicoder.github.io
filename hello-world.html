The "Hello, World!" Program

Your first program to print "Hello, World!" to the screen

Nearly every programming tutorial, starts with printing "Hello, World!" to the screen. This is a long-held tradition. The most likely historical explanation is simply that a short program like Hello World once allowed the programmer to make sure that a language's compiler, development environment, and run-time environment were correctly installed. Because this involved a lot of work, a very simple program was used first to test things out. According to Wikipedia, the tradition of using the phrase "Hello, world" as the test message was influenced by an example program in the book "The C Programming Language", by Brian Kernighan and Dennis Ritchie. To know more, do read the article on this at TechTarget from where this explanation was taken.

Type the following code in the java editor of your choice.

            
            class HelloWorld {
                public static void main(String[] args) {
                    System.out.println("Hello, World!");
                }
            }
